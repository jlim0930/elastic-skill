# Redaction Standards

All analysis output must strictly redact the following:
- **IP Addresses**: Replace with <ip> or <node-ip>.
- **Hostnames**: Replace with <hostname> or <node-name>.
- **Cloud IDs**: Replace with <cloud-id>.
- **Cluster IDs**: Replace with <cluster-id>.
- **Organization IDs**: Replace with <org-id>.
- **Passwords/Secrets**: Never include in any form.
- **Email Addresses**: Replace with <email>.
